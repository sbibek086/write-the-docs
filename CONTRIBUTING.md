# Contributing to this tutorial

## Update the files or documentation

Fork this repository and open a pull request with your changes.

## Add suggestions or bug reports

Add an issue for investigation.

## Adding Blog Posts with PDFs

If you want to add blog posts that include PDF files:

1. Add your PDF file to the `docs/pdfs/` directory
2. Create your blog post in the `docs/` directory
3. Embed the PDF using one of the methods described in [How to Embed PDF Files](docs/how-to-embed-pdf-in-blog-post.md)
4. Always include a download link as fallback

For detailed instructions, see:
- [PDF Embedding Guide](docs/how-to-embed-pdf-in-blog-post.md)
- [Example Blog Post](docs/2026-01-05-pdf-embedding-example.md)
